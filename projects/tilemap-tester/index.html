How to Use the Tilemap Editor

1. Set the Tile Size (in pixels) to define the dimensions of each tile in your tilemap.

2. Specify the Grid Size (width √ó height) to set the dimensions of your grid in tiles.

3. Click Update Grid to apply the tile and grid settings.

4. Toggle Show Grid to display or hide the grid overlay on the canvas.

5. Use the Color Picker below to change the application's background color.

6. Drag and drop a tilemap spritesheet (PNG, JPG, or GIF) into the Tilemap Spritesheet upload area.

7. Use the Grid Editor canvas to design your tilemap by placing tiles from the spritesheet.

8. Left click to place tile

9. Right click to rotate tile

10. Middle-mouse click to delete tile

8. Click Save to save your work or Load to retrieve a previous tilemap.
